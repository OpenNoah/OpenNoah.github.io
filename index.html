OpenNoah project

Welcome to the OpenNoah project page. GitHub

This is a fan project, mainly for reverse engineering the NP1xx0 series, hand-held embedded learning devices, designed by noahedu.com

Repository Description
linux Linux kernel from Ingenic (2.6.24.3)
linux-new The new Linux kernel (EXPERIMENTAL) (4.10.4)
mipseltools-gcc412-lnx26 Cross compile toolchain for mipsel, with gcc 4.1.2, for linux 2.6
mtd-utils Utilities for MTD subsystem (and UBIFS)
NoahSplit Program for decoding and splitting upgrade file
